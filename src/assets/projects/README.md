# Project Images

Add your project screenshots to these folders:

- `dues-reminder/` - Dues Reminder System screenshots
- `neonrail/` - NeonRail FPS game screenshots  
- `data-cleaning/` - Data Cleaning Assistant screenshots
- `data-scraping/` - Data Scraping & Email Automation screenshots
- `oil-gas/` - Oil and Gas Analytics dashboard screenshots
- `sales-insights/` - Sales Insights Dashboard screenshots

## Naming Convention
Use descriptive names like:
- `screenshot-1.png`
- `dashboard.png`
- `workflow.png`
